# python-expanse-tracker

Expense Tracker in Python 🐍 | CLI-Based Budget Management System

A Python CLI Expense Tracker that helps users track, manage, and summarize daily expenses efficiently.
This project uses CSV file storage and provides a menu-driven command-line interface for personal finance management.

Ideal for Python beginners, semester projects, and logic-building practice.

🔍 Project Keywords 

Python Expense Tracker, CLI Expense Manager, Budget Tracker Python, CSV Expense Tracker,
Python File Handling Project, Python Beginner Project, Command Line Application Python

✨ Features

➕ Add expenses with category, amount, description, and timestamp

📋 View expenses in a tabular format

✏️ Edit existing expense records

🗑️ Delete single or multiple expenses

🧹 Reset / format expense data safely

📊 Generate category-wise and total expense summaries

💾 Persistent storage using CSV files

⚠️ Input validation for reliable execution

🛠️ Tech Stack
Technology	Usage
Python 3	Core language

Datetime	Timestamp generation

🧠 How It Works

Automatically creates data.csv if not present

Uses a loop-based main menu for navigation

Stores expense records permanently

Supports partial editing (leave field blank to keep old value)

Generates real-time summaries from stored data
